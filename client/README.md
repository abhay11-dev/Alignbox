# Alignbox Client

This is a Vite + React frontend for the Alignbox chat backend.

Quick start:

1. Copy environment example:

```powershell
cp .env.example .env
```

2. Install dependencies:

```powershell
cd client; npm install
```

3. Run dev server:

```powershell
npm run dev
```

By default the dev server proxies `/api` and `/uploads` to `VITE_API_BASE`.
